decentralized android

  • September 2, 2018
  • tech  

言うまでも無くandroidは、Linuxと血を分けた妹である。
資本主義の暗黒面に魅入られてしまったandroidである。秘密裏に収集された個人情報が、どのように利用されるのかわかった物ではない。Googleが個人情報を分析し、私が消費行動を起こしやすいように広告が最適化される。その可能性が少なからずある以上、おちおちGoogleも使っていられない。
androidはこんな姿になってしまったが、根幹の部分はLinuxの血が流れている。オープンソースアプリをインストールすることで、Googleの鎖から解き放つことが目的だ。

まず第一にgoogle製のアプリを無効化する。余分なものがインストールされていないピュアなOSが売りのandroid oneも、産みの親の魔の手までは逃れられない。本来であればLineageOSなどをROM焼きするのが最善なのだが、買ったばかりの機種をroot化するのは怖い。
その後に広告のない、オープンソースのアプリをインストールする。F-droidを使うのがいいだろう。オープンソースのアプリを集めている。

なるべくandroidのアプリをオープンソースか広告のないものにしていた。ポイントはdecentralized時代の新しいandroidだ。
Chrome→Brave Browser
Gmail→Protonmail
Googleマップ→OpenStreetMap
Gdrive→NextCloud メッセンジャー→Signal
XMPPクライアント→Conversations
facebook→diaspora*
ファイルマネージャー→amaze file manager

このようにしてオープンソースアプリで武装することにより、androidは本来の姿を取り戻していく。厳密に言えば完全なオープンソース化にはこだわっていないが､基本的なツールにまで広告を貼るのは間違っている。
ちなみにマイクロブログサービスを使っていないのは、「ああいうツールはブログやホームページのサブウェポンとして使用するべきものである」という理由からだ。情報の蓄積性、検索性、文章の断片化などの問題があるので、短い情報を手っ取り早く伝えるにはいいがメインの言論活動ツールには適さない。やはり人類はブログかホームページだ。
