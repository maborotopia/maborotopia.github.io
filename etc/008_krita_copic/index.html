The Krita color palette of copic's 322 colors

  • May 3, 2019
  • tech  

This is the Krita color palette of copic’s 322 colors.
Compared to the default marker palette, This palette has the following excellent features.

・Many colors.
・Colors are organized into groups according to the Copic Color Theory.
・Using RGB approximate value data.

  • Copic Marker
  • Copic Color Theory | Copic
  • Too.com：コピック：ダウンロード

Download

copic322_color.gpl

  • maborotopia/Krita_resources

Install
Open the *.gpl file in Krita palettes docker.
This file is probably compatible with Gimp and Mypaint.
