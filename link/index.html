・役に立つツールなど

Privacy Tools
生き馬の目を抜く現代インターネット社会を生き残るための、プライバシー保護・セキュリティノウハウ。
Raspberry pi
シングルボードコンピュータの先駆者にして王者。現時点での最新機種はRaspberry pi 3B+。その小さな身体にクアッドコア1.4Ghzとメモリ1GBを持ち、想像以上にパワフルに働いてくれる。コストパフォーマンスと応用性に優れた素敵な機械。
Lubuntu
このブログでおすすめしているLinuxディストリビューション。軽量でネットブックでも動作するが、デザインがややダサい。その代わりに軽く、余分な機能を持たない質実剛健なデスクトップ環境だ。
Pocket
インターネットの記事を保存できるサービス。その真価は電子書籍リーダーkoboと連携したときに最大限発揮される。
koboにはpocket連携サービスが付属しており、英字新聞などの記事を電子書籍リーダーに送ってゆっくりと読むことができる。koboはサービス開始当初の炎上騒ぎで日本でのブランド価値に傷がついたが、そのせいかフリマアプリで比較的安価に中古端末が入手可能。
※ただ、日本語電子書籍の縦書きレイアウトがいまいち洗練されていない。かろうじて日本語が読める段階にとどまっていて、文字のフェチズムに乏しい。
Krita
高機能ペイントソフト。ブラシのカスタマイズ機能が豊富。
ProtonVPN: Secure and Free VPN service for protecting your privacy
無料プランでもそこそこ使い勝手のいいVPN。 美佳のタイプトレーナ

GitHub - featurecat/lizzie: Lizzie - Leela Zero Interface
AI囲碁ソフト。主に棋譜の検討に使う。
GitHub - lllyasviel/style2paints: sketch + style = paints
自動彩色ツール。自分でローカル環境にサーバーを建てないといけないので少し敷居は高い。

フリーフォント
源ノ明朝
　GoogleとAdobeの開発したフリー日本語フォント。太さの種類が多く、ディスプレイでの表示もきれいだ。それ以外にもgoogleのweb fontsはフリーでありながらも豊富なデザインの書体が揃っている。 linuxにインストールするときには sudo apt-get install google-noto-cjk
源流明朝
源雲明朝
源暎こぶり明朝
　源ノ明朝派生フォント。源雲明朝はにじみ処理がお気に入りで、源暎こぶり明朝は縦書きの文章を印刷するときに使う。
Google Web fonts
　フォントを買う金のない奴でも使える。有名フォントに似た書体も一通り揃っている。でもオリジナルを作ったデザイナーの顔に泥を塗っているような気になる。ちなみに好きなフォントはDINとCopperplate gothicと、Garamondです。
