Raspberry pi
シングルボードコンピュータの先駆者にして王者。現時点での最新機種はRaspberry pi 3B+。その小さな身体にクアッドコア1.4Ghzとメモリ1GBを持ち、想像以上にパワフルに働いてくれる。コストパフォーマンスと応用性に優れた素敵な機械。

※ただ、日本語電子書籍の縦書きレイアウトがいまいち洗練されていない。かろうじて日本語が読める段階にとどまっていて、文字のフェチズムに乏しい。

Linux ツール

すべて sudo apt-get install hogehoge　でインストールできる。
redshift
画面のブルーライトをカットして、目に優しい色に変えてくれる。時間帯に合わせて画面の色温度を変更し、眠る前には赤みがかった刺激の少ない色になる。
aria2
並列ダウンロードをしてくれる機能。sudo apt-get install aria2　で導入し、aria2c -x5 hogehoge　で高速で並列にダウンロードする。大きなファイルを落とす時に役立つ。
fdupes
重複したファイルを削除するツール。
nkf
文字コードを変更するツール。Windowsのshift-jis形式をUTF-8に変換する時によく使った。
ebview
Linuxでの数少ないEpwing形式辞書ビューア。
cherrytree
アウトラインエディタ
Leafpad
シンプルなテキストエディタ。lxappearanceのAbwaita-darkというテーマを使うと背景をダークグレーにできて目に優しい。様々なテキストエディタを試して、最後にはシンプルなものに帰ってくるのだった。
lxappearance
デスクトップの見た目を変える。使っているデスクトップはlxdeとかpixelとかopenboxだ。
okular
快適に読めるPDFビューア。メモリ使用率をGreedyに変えるとメモリを貪欲に使用してPDFを先読み表示する。
Recoll
デスクトップ全文検索ソフト。インデックスを作成するので素早く検索できる。

フリーフォント 源ノ明朝
　GoogleとAdobeの開発したフリー日本語フォント。太さの種類が多く、ディスプレイでの表示もきれいだ。それ以外にもgoogleのweb fontsはフリーでありながらも豊富なデザインの書体が揃っている。 linuxにインストールするときには sudo apt-get install google-noto-cjk

源雲明朝
源暎こぶり明朝
　源ノ明朝派生フォント。源雲明朝はにじみ処理がお気に入りで、源暎こぶり明朝は縦書きの文章を印刷するときに使う。
Google Web fonts
　フォントを買う金のない奴でも使える。有名フォントに似た書体も一通り揃っている。でもオリジナルを作ったデザイナーの顔に泥を塗っているような気になる。ちなみに好きなフォントはDINとCopperplate gothicと、Garamondです。
