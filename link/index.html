Pocket
インターネットの記事を保存できるサービス。その真価は電子書籍リーダーkoboと連携したときに最大限発揮される。
koboにはpocket連携サービスが付属しており、英字新聞などの記事を電子書籍リーダーに送ってゆっくりと読むことができる。koboはサービス開始当初の炎上騒ぎで日本でのブランド価値に傷がついたが、そのせいかフリマアプリで比較的安価に中古端末が入手可能。
※ただ、日本語電子書籍の縦書きレイアウトがいまいち洗練されていない。かろうじて日本語が読める段階にとどまっていて、文字のフェチズムに乏しい。

Linux ツール

すべて sudo apt-get install hogehoge　でインストールできる。
redshift
画面のブルーライトをカットして、目に優しい色に変えてくれる。時間帯に合わせて画面の色温度を変更し、眠る前には赤みがかった刺激の少ない色になる。
aria2
並列ダウンロードをしてくれる機能。sudo apt-get install aria2　で導入し、aria2c -x5 hogehoge　で高速で並列にダウンロードする。大きなファイルを落とす時に役立つ。
fdupes
重複したファイルを削除するツール。
nkf
文字コードを変更するツール。Windowsのshift-jis形式をUTF-8に変換する時によく使った。
ebview
Linuxでの数少ないEpwing形式辞書ビューア。
cherrytree
アウトラインエディタ
Leafpad
シンプルなテキストエディタ。lxappearanceのAbwaita-darkというテーマを使うと背景をダークグレーにできて目に優しい。様々なテキストエディタを試して、最後にはシンプルなものに帰ってくるのだった。
lxappearance
デスクトップの見た目を変える。使っているデスクトップはlxdeとかpixelとかopenboxだ。
okular
快適に読めるPDFビューア。メモリ使用率をGreedyに変えるとメモリを貪欲に使用してPDFを先読み表示する。
RedNotebook
日記ソフト。
Recoll
デスクトップ全文検索ソフト。インデックスを作成するので素早く検索できる。

英語学習リソース

The Oxford 3000
　英単語帳にはTOEIC頻出順単語だとか、大学入試用といった目的に分かれているのだけれども、oxford 3000は「実際に使われている英文を統計分析して、頻出する単語をリストにしたで。外国の英語学習者は参考にしてなー」というものだ。
Roget’s 21st Century Thesaurus, Third Edition
　英語のシソーラス。知らない言葉に触れるとわくわくするよね。
English Grammar in Use Book with Answers and Interactive
　英文法の参考書。洋書だが平易な英語でわかりやすい。
Word Power Made Easy: The Complete Handbook for Building a Superior Vocabulary
語源を覚えてボキャブラリーを増やそう！という本。
「英語で書く」基本が身につく本
　プレイン・イングリッシュの本。難しい単語を使わずに、平易な構文と表現で英文を書くための方法について触れられている。
　日本語で書いた文章を小文に分解し、翻訳した英文をつなぎ合わせるという手法は、翻訳ソフトを使用するときにも役に立つ。長文をそのまま翻訳させると意味の通じない日本語になってしまうが、短い文章ならうまく翻訳してくれる場合が多い。
コリャ英和!一発翻訳2019 for Win
　海外のニュースサイトなどを読むときに、「訳振りパッド」が役に立つ。辞書を引く手間を大幅に削減し、英文を読む負担を軽減してくれる魔法の機能だ。が、未だに使っているのは2012年版だ。
