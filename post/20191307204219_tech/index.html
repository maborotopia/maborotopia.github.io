・Fluid motion

Ryzen Gシリーズには24fpsの動画を60fpsに補完してくれるFluid motionという機能がある。
これは静止画を高速でスライドさせる演出が多い映像と相性がいい。つまり低予算アニメとギャルゲーOPである。これまでは24～30fpsの壁に阻まれてガクガクしていた映像もFluid motionを使えばなめらかな映像になる。すごい！　これ無しでもうアニメ観たくない！

・テクノロジー左翼日記。

テクノロジー左翼なのでオープンソースソフトウェアを中心に使う。
Google Playストアのアプリは広告まみれなので、心底嫌気がさしてしまった。androidで音声レコーダーをインストールしたのだが、基本的なツールにまで広告が表示される。Ad-Freeなアプリを探す方が難しい。この惨状を見て私は悟った。Googleは広告代理店で、androidは我々に広告を見せるために作り出されたOSだ。Googleのエコシステムにとどまる限り、広告から逃れられない。
F-droidでオープンソースソフトウェアをインストールすれば広告地獄からは逃れられるのだが、完全なオープンソースのスマートフォンにするのは茨の道だ。
脱GAFAのスマートフォンを作るのにも様々な流派がある。まず第一はGoogleと名の付くものであればすべて排除しようとする原理主義集団だ。android端末を便利に使うためにはGoogle開発者サービスが必要なのだが、そのインストールを拒絶してオープンソースソフトウェアのみを使う。
問題はGoogle開発者サービスなしで動くオープンソフトウェアは数少ないということだ。
たとえば堅牢なセキュリティで知られるProtonmailのandroid版はGoogle開発者サービスが無いと通知機能が働かなくなる（らしい）。Signalメッセンジャーも同様だ。それを避けるためにはXMPPやTutanotaなどのサービスに頼らなければならない。
最初は脱Google原理主義者として生きようと思ったが、挫折してしまった。LineageOSをROM焼きして、端末の保証は失われ、まっとうに生きている日本人なら聞いたことが無いようなサービスを使い、シェアの少ないメッセンジャーアプリを使う。使うSNSはdiaspora*とMastodonだ。
ここまでやると人間世界との接点を失ってしまう。「快適さを犠牲にしてでもオープンソースソフトウェアで統一しなければならない」と思い、イデオロギーを優先して現実の使い勝手を無視する。ゆがんだ共産主義理論を押しつけて国家に深い傷を残したクメール・ルージュみたいな危険思想OSであり、もはやテクノロジー極左だ。
第二の道は妥協だ。GAFAから完全に逃れるのではなく、中途半端に妥協してGAFAへの依存を減らすというものだ。AndroidはそのままでGoogleにもログインしている。代替サービスを上手に利用し、使い勝手とプライバシーのバランスを探る。これはテクノロジー中道左派である。
ひとくちに脱GAFAといっても様々な道があるのだが、にんげんは思想と実用性の間に折り合いを付けてなんとかやっていくしかないのよね……と思い知らされた。現実をそのまま受け入れるわけにはいかないが、思想が先鋭化すれば現実が犠牲になる。

・Ryzenの内蔵GPUは性能がいい。

Ryzen 3 2200GはGPUを導入しなくてもやFF14やスカイリムなどのゲームが動く（らしい）。
最新のPCゲームをするつもりがないし、ちょっとした軽いタイトルが動けばいい。内蔵GPUに多くは求めない。遊びたくなったらそのときにはちゃんとゲーミングPCを組み立てる。そう思って選んだCPUだったが、思いの外グラフィックが高性能だった。これはうれしい誤算だ。

  • Ryzen Desktop APUs benchmarked: Destroy Intel Core i5-8400 in 1080p Gaming

  • 【レビュー】【本日発売】GPU内蔵になったRyzenの性能をベンチマーク - PC Watch

  • AMDがRyzen 3000シリーズの国内価格を発表　8コア16スレッドのRyzen 7 3700Xは税別3万9800円 - ITmedia PC USER

第三世代のRyzenが発表されたが、GPU内蔵モデルに関してはあまり魅力的なスペックでは無かった。てっきり6コア12スレッドだとか、コア数は据え置きだがRadeon RX Vega 19搭載!　完全にエントリー向けグラボを越える！みたいな展開を夢見ていた。
あと第二世代のRyzen 5 2400Gが44%オフになっていた。

Ryzen Gシリーズはコストパフォーマンスがいい。これは「コスパが良すぎるあまりに自社製品と共食いを始める」というマックチキン現象の再来である。かつてマクドナルドの100円メニューにはマックチキンがあり、300円台のバーガーに迫る実力を持っていた。100円バーガーの貧相な肉とは比べものにならない食べ応えがあり、財布にも優しい。しかしそれ故に販売休止になり、チキンクリスプという形で味が下方修正されたという苦難の歴史を持つ。
Ryzen 3 2200Gを見ているとマックチキンの面影を思い出してしまう。もしも6コア6スレッド・99ドルなんてことになれば、バランスブレイカーとして市場に混迷をもたらすことは不可欠だ。

・連絡用メモ

下記のような性質があるので、SNSメディアやスマートフォンをあまり使わないようにしている。ご不便ですが、ご協力のほどよろしくお願いします。

・フリック入力ができないので、lineとスマホが使えない。
フリック入力ができない。不可能なわけではないのだが、キーボードに比べると入力に手間がかかるので可能な限り使いたくない。

・部屋に人間関係を持ち込みたくない。
SNSやスマートフォンを使っていると、部屋の中にいても対人関係処理モードに入るので精神的に消耗してしまう。なるべく家の中にいるときにはスマホを触らないようにしたい。

チャット的なやりとりがきつい。
メッセンジャーアプリでの「いま暇？」→「うん」→「ひさしぶりだね～」→「最近何しているの？（以下延々とつづく）」というようなチャット的なやりとりをしないようにしています。

急に通知音や着信音が鳴ると驚く
聴覚過敏なので、急にスマホの着信音が鳴ると心臓に悪い。

・連絡先をメールボックスに一本化した。リアルタイムでの連絡は取れない。

・閉鎖的なプラットホームは悪。
・チェックする受信ボックスを増やしてはならない。
・通知でアテンションが削られるのを避ける。
・特定の機器でしか使えないサービスを使わない。
上記の条件を満たした結果、どんな環境でも使える電話とメール最高！という結論になった。
さまざまなSNSサービスをまとめて管理できるアプリもあるが、物事を複雑にしすぎるのは何かが間違っている。そもそもユーザーにITリテラシーの差があるので、導入が面倒くさいツールや日本語化されていないサービスは使えない。
ブラウザに拡張機能を追加してSNSにアクセスする方法もあるのだが、サービスごとにアカウントを取得するのが面倒くさいという理由により却下。

IP電話を使うと着信履歴と伝言メッセージをメール宛てに添付できる。これで受信ボックスと電話を統合して、外出するとき以外はスマホの電源をオフにする。チェックは一日に数回すればいい。それで連絡が取れないのなら諦める。
スマホを使うのは待ち合わせや急な連絡が必要なときに限る……という方向でデジタルガジェットを制限していた。

・2019年、RSSリーダーの帰還

RSSリーダーを再び使い始めた。Googleリーダーが終了した後にはFeedlyなどを転々としていたのだが、更新するブログも少なくなっていってRSS自体を使わなくなってしまった。しかし自分の見ている情報源が日を追うごとに狭くなっているので、RSSリーダーを復活させて、このインターネットの海から無限の情報を引き出すのだ。
網羅的に情報を得るためでは無くて、SNSでは埋もれがちになってしまう記事を取りこぼさないためにRSSを使った方がいいのかも知れない。数日置きに更新される記事は、更新間隔の短いユーザーの投稿で埋もれてしまう。
「タイムラインがひとつだけでは更新間隔が異なったメディアを上手く管理できない。流速ごとにフォルダ分けをしたほうがいい」という考えに基づいて、RSSフィードを管理するようにしたよ。
あとは「２４インチのディスプレイにびっしりと小さな文字が映し出されるのが、気持ちよくてたまらねえぜ、ひゃあっ！」というテキスト・フェチズムを刺激したい。５インチ程度の画面をスクロールし続ける日々は今日で終わりだ。

・最新と最古の間で。

囲碁ソフト・Leela Zeroの性能を発揮するためには、高性能のグラフィックボードが求められる。性能のいいグラフィックボードを積むと速度が10倍、20倍になり、リアルタイムでGPUをぶんまわして着手の善し悪しを検討できる。
だが、そのためには最新のゲームがスムーズに動くようなゲーミングPCを手に入れる必要がある。囲碁にゲーミングPCが必要なのか？と思ったが、そもそも盤上ゲームだった。ゲームをするためにゲーミングPCを手に入れるのは道理にかなっている。
そのうちAMDの新アーキテクチャGPUも出るので、更に性能は上がるはずだ。

囲碁の棋譜を検討するのにグラフィックボードに気を遣わなければならないとは思わなかった。そのうち囲碁老人たちが「AMDの新型グラボ欲しい」と言い始めたり、NHK囲碁講座で「自作PCで高速マシンを作る！　コスパ重視マシンと、金に糸目をつけない高性能マシン！」とかDOS/Vパワーレポートのような特集を始めたりしかねない。 囲碁が趣味の高齢者には余裕のある高所得者世帯が多いという。何十万もする碁盤の通販広告が載っているから、20万円ぐらいするグラボも売り込めるはずだ。

囲碁は枯れた趣味だ。ゆっくりとやろう。昭和時代の棋士を参考にしながら頑張るぞ。そう思いながらネット碁を打っていた。読んでいる布石本の発行日は1988年ぐらいで、何十年も前の古びた知識で異国の人々と渡り合っていた。
「今日は江戸時代に活躍した本因坊秀作の棋譜を見て、厚い打ち方を学ぼう」と思っていたのだが、時は人工知能時代である。Githubからソースコードを落として、囲碁AIをビルドしていた。江戸時代とGithubという異質な単語が、囲碁を介してつながる。その時間感覚の落差に混乱している。

・情報拡散の不健全なビジネスモデルについて。

Twitterはインターネットの健全性を阻害している。社会のインフラであるような素振りを見せているが、やつらのやっていることはインターネット文化大革命だ。
一部の有名人やアルファユーザーをのぞいてわざと情報が拡散しにくくなる仕組みを作っている。そのうえで「あなたに宣伝したいことはありますか？　そのためにはプロモーションに金を払って下さい」とマネタイズにつなげるのは倫理的では無い。情報拡散が企業の私有物になっている。
広告を売る側からすれば、自力で情報が拡散しにくい環境の方が好ましい。情報が細切れになって、過去ログが閲覧しにくくなって、コンテンツへの導線が細くなればなるほど、金を払って広告を打つ価値が上がっていく。
金を払わないやつは検索エンジンに埋もれてしまうし、それが嫌ならプロモーションに資金を費やせる組織が勝つ。IT企業が作ったプラットフォームの中で、必死になってフォロワーを増やしたり、情報を拡散させようとするのはナンセンスなことでは無いのか？と思った。
ソーシャルゲームでは、無課金ユーザーが非効率的な方法を強いられる。その苦痛を金を払ってスキップするのがビジネスモデルだが、情報を拡散する行為にもソーシャルゲームと似たビジネスモデルになっている。

・パソコンから日本語フォントを削除する話。

パソコンから日本語フォントを削除して、ゆがんだ言語環境から逃れる方法について書いていく。
何も考えずに日本語のインターネットにつないでいると、嫌な気持ちになることが多くなった。ヘイトスピーチや誹謗中傷だけではなくて、常に何かしらの負の感情が刺激される。犯罪者への言及、差別的な言説、他者の不道徳な発言、マナー違反、フェイクニュースなどに反応してしまって、無駄なところで感情エネルギーを摩耗させている。
むやみやたらに感情を刺激される場所から逃れたいと思ったので、デスクトップPCから日本語フォントを削除している。これでなんとなくブラウザを開いて、なんとなくSNSのトレンドを見て、誰かの感情に巻き込まれて不快な気分にならずに済む。

・システムから日本語フォントを削除する。
Lubuntuの場合は下記のコマンドラインを使い、すべてのフォントを削除したあとにシステムフォントのみをインストールしている。
sudo apt-get purge -y fonts-* ; sudo apt-get install -y fonts-ubuntu

・次にテキストエディタで下記のファイルを編集し、ウェブサイトにアクセスできないようにする。/etc/hosts は指定したサイトへのアクセスを制限するためのファイルだ。
sudo nano /etc/hosts

それでもコマンドラインから簡単に日本語フォントをインストールできてしまうので、/etc/apt/sources.list を編集してコマンドラインからソフトウェアのインストールができないようにする。
sudo nano /etc/apt/sources.list

・仮想デスクトップに他のOSをインストールし、日本語入力環境を確保する。仮想デスクトップをオフライン状態にすれば、外部環境から隔絶できる。
なぜかLubuntuにWindows XPを仮想デスクトップ化して擬似ゼロ年代環境を作り上げている。魂がまだゼロ年代に留まったままなので、SNSには適応できないのだ。
・通信手段はスマホに集約する。
・どうしても日本語を読まなければならないときには、google翻訳を噛ませてから英語で読むようにする。そうすることで日本語に直接触れないようにする。
・なるべく活字の文章を読むようにする。
・これらの行為を駆使してネット上の日本語をスルーすると、不用意に感情を刺激されずに済むのですごい楽で快適になる。やったね！

ここまでやる必要があるのかどうかわからないのだが、ネット上の日本語に触れることそのものがストレス源になってしまったので仕方が無い。日本語環境から距離を置くことでしか苦痛を和らげられない。
加工されていない生の感情がなだれ込んでくるような感覚になるので、ネット上の日本語をまともに受け止めるのは疲れる。情報は少なくて、反対に感情過多になっている。腐海で胞子を吸い込まないようにマスクをするみたいに、言葉そのものをフィルタリングした方が精神衛生を保てる。
